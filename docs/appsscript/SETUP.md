# 備品「移動」書き戻しのセットアップ（1回だけ）

キオスク画面の移動をスプレッドシートへ書き戻す設定。サーバー不要・無料・5分ほど。

> **オーナー権限は不要。スプレッドシートの「編集者」権限があればOK。**
> ウェブアプリを「実行：自分」でデプロイすると、あなた（編集者）として動き、
> あなたの編集権限で書き込むため。コードは ID 指定でシートを開くので所有者でなくて良い。

---

## ⚠️ 最重要：コードは「ファイル全体」を置き換える

エディタに最初からある `function myFunction() { … }` を**含めて全部削除**し、
[`Code.gs`](./Code.gs) の中身を**まるごと貼り付け**ます。
正しい状態＝ファイル先頭が `/**` で、`function doGet` と `function doPost` が
**いちばん外側（トップレベル）**にあること。
※ `myFunction(){ }` の“中”に貼ると動きません（よくある失敗）。

---

## 方法A：独立スクリプト（非オーナー／編集者におすすめ）

オーナーでなくても自分の権限だけで完結します。

1. https://script.google.com を開く → **新しいプロジェクト**
2. 既定の `myFunction` を含め全消し → `Code.gs` をまるごと貼り付け → 保存（💾）
3. 右上 **デプロイ → 新しいデプロイ**
   - 種類（⚙）：**ウェブアプリ**
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**
   - **デプロイ**
4. 権限の承認（「確認されていません」→ 詳細 → 続行）。
   スプレッドシートへのアクセス許可を求められたら許可
5. 出てきた **ウェブアプリ URL（…/exec）** をコピーして Claude に渡す

## 方法B：スプレッドシート内蔵スクリプト

1. スプレッドシートで **拡張機能 → Apps Script**
2. 上の「⚠️最重要」のとおり**ファイル全体を置き換え**て保存
3. 以降は方法Aの手順3〜5と同じ

> どちらでも結果は同じ。1つ動くURLがあれば十分です。

---

## 動作確認（任意）

- 取得したURLをブラウザで開くと `{"ok":true,"overrides":{}}` が出ればOK
- スプレッドシートに「移動記録」タブが自動作成される（元の部屋別タブは無変更）

## 補足（Workspace制限の場合）

会社/学校の Google Workspace アカウントだと「アクセス：全員（匿名含む）」が
管理者設定で選べないことがあります。その場合は個人Googleアカウント
（スプレッドシートに編集者で招待されたもの）で方法Aを実施してください。

## 仕組み

- 画面で移動 → このウェブアプリへ送信 → 「移動記録」タブに `id 単位で上書き`保存
- 画面は約1分ごとに現在の移動状況を取得して全端末で同期
- 操作端末は即時反映＋数秒間「取り消す」可能
- 元データ（部屋別タブ）は無変更。`scripts/importItems.mjs` は「移動記録」を読まない
