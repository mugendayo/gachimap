# 備品「移動」書き戻しのセットアップ（1回だけ）

キオスク画面から備品を移動したら、その情報をスプレッドシートに自動で書き戻すための設定です。
サーバー不要・無料。所要 5分ほど。

## 手順

1. 対象のスプレッドシートを開く
   （https://docs.google.com/spreadsheets/d/1-Vbh4A2LPBDxn2m8xrQTrSEJzKqv36sDZ4_-pjtJ0mw/edit）

2. メニュー **拡張機能 → Apps Script** を開く

3. 既定の `Code.gs` の中身を全部消し、本リポジトリの
   [`docs/appsscript/Code.gs`](./Code.gs) の内容を丸ごと貼り付けて保存（💾）

4. 右上 **デプロイ → 新しいデプロイ**
   - 種類（歯車）: **ウェブアプリ**
   - 説明: 任意（例: gachimap move）
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**
   - **デプロイ** をクリック

5. 初回は権限の承認を求められる → 自分のGoogleアカウントで許可
   （「このアプリは確認されていません」と出たら **詳細 → （安全でないページ）に移動** で進む）

6. 表示される **ウェブアプリの URL**（`https://script.google.com/macros/.../exec` で終わる）をコピーして
   Claude に渡してください。サイト側に接続して再デプロイします。

## 動作確認（任意）

- URL をブラウザでそのまま開くと `{"ok":true,"overrides":{}}` のような JSON が出ればOK。
- シートに「移動記録」タブが自動作成されます（元の部屋別タブは変更されません）。

## 仕組み

- 画面で移動 → このウェブアプリへ送信 → 「移動記録」タブに `id 単位で上書き`保存
- 画面は約1分ごとに現在の移動状況を取得して全端末で同期
- 操作した端末はその場で即時反映（楽観更新）＋数秒間「取り消す」可能
- 元データ（部屋別タブ）は無変更。`scripts/importItems.mjs` は「移動記録」を読まないので影響なし
